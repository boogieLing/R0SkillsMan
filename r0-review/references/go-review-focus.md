# Go Review Focus

## 并发与生命周期
- goroutine 是否可退出，是否可能泄漏。
- channel 关闭时机是否明确，是否存在重复 close。
- `context.Context` 是否贯穿下游调用并正确传递取消信号。

## 错误处理
- 错误是否被吞掉（`_ = err`、仅打印不返回）。
- 是否使用 `errors.Is/As` 进行语义判断。
- 日志与返回错误是否有足够上下文（请求 ID、关键参数）。

## 资源与性能
- `defer` 在循环内是否造成额外开销或资源积压。
- 热路径是否有不必要分配（string/byte 反复转换）。
- map 并发读写是否安全。

## API 与注释
- 导出符号是否有 GoDoc，注释是否与实现一致。
- 接口语义变更是否破坏兼容性。
