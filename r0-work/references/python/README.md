# Python Engineering References (r0-work)

This directory defines **mandatory engineering constraints** for Python projects
executed under the `r0-work` Codex skill.

These documents are NOT suggestions.

They define **hard constraints** that MUST be followed whenever:
- Project language is Python
- Runtime environment is Python 3 (Conda-based)

Violating any rule defined here is considered a **skill violation**.

---

## Environment Assumptions

- Python 3 ONLY (Python 2 is strictly forbidden)
- Conda / Miniconda based local environment
- Modern Python tooling is assumed (typing, dataclasses, etc.)
